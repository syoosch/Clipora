# 更新日志

本文件记录 Clipora 每个版本的变更。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循[语义化版本](https://semver.org/lang/zh-CN/)。`0.x` 为 1.0 前的公开预览，功能可能随反馈调整。

## [0.4.4] - 2026-07-21

### 修复
- 图片预览改用统一状态机管理悬停、异步加载和关闭：滚动、窗口移动/缩放、拖拽、失焦等场景会立即取消陈旧预览，避免虚拟化容器回收后崩溃或错误弹出。
- 修复 125% 及更高 DPI 下大图预览与主面板重叠；定位统一使用设备坐标，并覆盖多显示器、负坐标、小工作区和极端缩放。
- 修复光标停在卡片下描边附近时，3 DIP 上浮动画导致卡片反复进入/离开并上下抽搐；固定命中面与底部保留带保持原动画和卡片间距。

### 优化
- 图片预览要求连续悬停 350ms；滚动停止 120ms 后才重新计时，滚动期间不再抢先弹出预览。
- 大图预览采用稳定的 400×400 DIP 布局，优先放在主面板外侧并保留 12 DIP 间距；空间不足时再转入面板内部。
- 图片解码移至单独 STA 工作线程，并使用有上限的本地缓存与最新请求优先队列，减少快速浏览图片历史时的 UI 干扰。

## [0.4.3] - 2026-07-13

### 安全
- 将 `SQLitePCLRaw.bundle_e_sqlite3` 显式升级并固定为 3.0.3，原生 SQLite 更新至 3.50.4.5，移除 `SQLitePCLRaw.lib.e_sqlite3 2.1.11` 及其 `NU1903` 高危漏洞告警。
- `.clpbak`、manifest、SQLite 与 payload 全部按不可信输入处理，阻断路径穿越、伪造 journal 越界删除、恶意 schema/行数据和归档声明不一致；合法 v1 备份保持兼容。
- 可执行、脚本、快捷方式等主动文件打开前默认取消并要求明确确认；URL 仅允许 HTTP/HTTPS，普通文件仍一键打开。
- 系统隐私标记无法确认时改为一次 50ms 非阻塞重试后跳过，避免异常时误捕获；Release 崩溃诊断改为本地脱敏、限期/限量保留。
- 新增双语隐私说明，明确数据库、附件和备份当前未加密；无需迁移现有剪贴板数据。

## [0.4.2] - 2026-07-05

### 修复
- **图片预览 200% DPI 过早关闭**：安全区右/下边界使用 DIP 尺寸直接相加，200% DPI 下缩略图实际物理宽高被减半，导致光标在缩略图内仍被误判为离开。修复：保留 cursor 和 `PointToScreen` 为屏幕坐标，将 `ActualWidth`/`ActualHeight` 与边距按 `TransformToDevice` 扩展为物理像素后比较。
- **移除 `Window.Deactivated` 关闭预览**：安装版环境下 Popup（`WS_EX_LAYERED`）会触发主窗口失焦 → 刚打开的预览被瞬间关闭。已移除该路径，仅保留 `GetCursorPos` 轮询 + `MouseLeave` 双重守卫。

### 优化
- 预览关闭容差调优（450→200→80px），减少屏幕边缘误触发。
- 预览安全区改用 Popup 实际渲染区域计算，替代 DIP 尺寸估算。

## [0.4.1] - 2026-07-04

### 修复
- **图片卡片悬停预览闪烁**：CliporaCard 悬停上浮动画（3px）将缩略图顶出光标范围导致虚假 MouseLeave → 自持闪烁循环，主面板靠近屏幕边缘时更明显。修复：`Thumbnail_MouseLeave` 增加 4px 容差检测过滤卡片动画虚假 Leave；Popup 打开后改用 `GetCursorPos` 屏幕绝对坐标轮询 + `Window.Deactivated` 失焦关闭，绕过 `AllowsTransparency="True"`（WS_EX_LAYERED）窗口的鼠标事件干扰。

## [0.4.0] - 2026-07-02

### 新增
- **卡片操作按钮**：卡片右上角 Pin 左侧新增类型感知操作按钮。文字/富文本/代码/颜色 → 内嵌覆盖层（可选取文字、字数统计、等宽字体）；图片/链接/单文件 → 系统默认打开（已保存文件设只读防覆盖备份）；多文件/文件夹不显示按钮。
- **设置页 info 图标**：新建 `SettingsRowControl`，标题旁 `ⓘ` 图标 hover 弹出 CliporaCard 风格浮窗、click 展开行内描述。全部 6 个分类中 5 个已迁移（共 17 行），外观分类无描述无需改动。

### 修复
- **FileNameTrimming 扩展名截断**：`Measure()` 使用的 `FormattedText`（GDI+ 遗留引擎）与 WPF 渲染文本宽度存在偏差，导致二分截断结果溢出、末尾扩展名被遮挡。改用独立 `[ThreadStatic]` TextBlock 测量 + 安全网二次核验与逐位退回。后续修复间歇性失效：测量块补全 `LineHeight`/`LineStackingStrategy` 克隆，`Apply` 延迟到布局完成后执行消除时序竞态。新增 15 文件名 × 280 宽度扫描自检。
- 备份标题多余省略号。

## [0.3.0] - 2026-07-01

### 新增
- **液态玻璃通透度滑条**：设置页新增 0–100 通透度滑条（默认 55），实时调节 Acrylic 背景透视感。
- **全窗口遮罩层**：液态玻璃主题下全窗口覆盖半透明遮罩，滑条调节的不再仅限于卡片，而是整个窗口对桌面的透视程度。

### 修复
- 启动时从 Fluent 切到液态玻璃瞬间闪退（`ColorConverter` 解析非法颜色 `##00FFFFFF` 崩溃）。
- 便签主题入口按钮移除，外观主题收敛为 Fluent / 液态玻璃二选一。

### 优化
- 拖动通透度滑条时不再闪烁（`SyncGlassOverride` 先 Add 后 Remove）。
- 滑条存盘采用 500ms 防抖（`DeferredSave`），避免高频写入。
- 协作流程分级（微改快速通道 / 标准小步 / 高风险完整模式），降低不必要流程开销。

## [0.2.0] - 2026-06-24

### 新增
- **液态玻璃外观主题**：第二套外观皮肤（强通透 Acrylic 质感），支持即时热切换；窗口失焦时保活渲染，避免背景发灰。
- **Fluent 主题自定义背景**：选本地图片作窗口背景，0–100% 透明度调节，按图片中心裁切铺满。
- **悬浮球拖入即存**：最小化为贴边悬浮球时，拖内容到贴边条即膨胀成方块、松手直接入库，主面板不打开。
- **图片文件拖入显示缩略图预览**：拖入的图片文件仍归类为「文件」（可拖出原文件、归在「文件」筛选），但卡片显示图片缩略图、可悬停放大。

### 修复
- 外部拖入浮层卡死：拖文件到桌面回收站等处时误触发主面板拖入提示并卡住；窗口内移动时偶发闪烁。
- 跨零点日期分组错误：应用跨过午夜后，次日新增内容被错误并入前一天的「凌晨」时间段。
- 单行文件名省略号会连扩展名一起截掉；现末尾扩展名（如 `.md`）始终可见。

### 优化
- 外观设置页选项按钮：平衡选中 / 未选中状态的悬停反馈。
- 自定义背景的「更换 / 清除」按钮：悬停高亮限定到按钮自身边界。
- 透明度滑条改为简洁的中性轨道 + 圆球。

## [0.1.0] - 2026-06-23

首个公开预览版。

### 新增
- 自动捕获并分类剪贴板内容：文字 / 富文本 / 链接 / 代码 / 颜色 / 图片 / 文件；也可把文件、文字、图片直接拖入面板保存。
- 按时间倒序、每天分凌晨 / 日间 / 晚间三段可折叠分组浏览；置顶区固定在顶部；卡片显示来源应用。
- 搜索与按「类型 / 标签」筛选；标签管理（贴标签、快捷新建、改名 / 删除 / 调色 / 排序）。
- 单条操作：置顶、删除（进回收站可恢复）、点击重用（复制并自动粘回原窗口）、拖出到其他程序。
- 托盘常驻 + 全局快捷键 Alt+V 呼出 + 自动粘贴；关闭收起到托盘；单实例运行。
- 最小化为贴边悬浮条（鼠标靠近呼出预览）；窗口置顶开关；不占任务栏。
- 数据保留期自动清理；内容超限提示、隐私应用排除等设置；Win11 云母质感、跟随系统深浅色。
- 纯本地存储，绝不联网、不上传剪贴板数据。
